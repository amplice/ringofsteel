## Intent

- `defense/block`
  - muted but still readable guard contact
- `defense/parry`
  - bright precise deflection
- `defense/clash`
  - larger collision event
- `hit/light`
  - crisp confirm with little tail
- `hit/heavy`
  - heavier contact with more body / resonance
- `hit/thrust`
  - narrower impact or stab accent
- `movement/footstep`
  - hard-surface and armored-step candidates for the arena

## Read

- `defense` is now built more around real weapon-on-weapon contacts, not generic metal pings.
- `hit` mixes:
  - more realistic sword-contact files
  - a few stylized heavier options
  - stab-specific options for thrust
- `footstep` now includes:
  - hard stone/concrete
  - metal-armored variants

If these still are not strong enough, the next step is not another blind sweep. It is:
- pick the best `1-2` in each folder
- wire them in
- judge them in-game against the current visuals and timing
