# Itch Pass Sound Review

This pass is narrower than the first one. It focuses on attack starts because the earlier shortlist was not good enough there.

Sources in this pass:
- `Swords & Blades Sound Pack` by Leonardo Calvo: explicit `CC BY 4.0` on itch page.
- `Whoosh Sound Effects` by Tagirijus: explicit `CC BY 4.0` on itch page.
- `Sword Combat Sound Effects Pack Free Version` by Hove Audio: useful pack, but not a standard open license. The itch page presents it as royalty-free / no-credit-needed. I marked those files separately so you can reject them if you want to stay strictly CC-only.
